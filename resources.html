Resources:

Learning to use machine tools is a long process, and anyone getting started on this road needs all the help they can get! For that reason I've created this page which lists links to websites that I find informative or inspirational. There are also links to useful suppliers.

Informative and Inspirational sites:

  • CNC Zone: Forum for discussion of CNC and manual wood and metal-working machines
  • Frets: Lots of neat projects for around the shop
  • 5 Bears: A homemade CNC mill, jet engines and more
  • Tooling Around: General info on small machine tools
  • Home Shop Machinist: A great source of machining information
  • Home Model Engine Machinist: Great for information on close-tolerance machining of small parts

Useful videos:

  • How to make a silicon mold
  • Machine Shop Tips: More than 80 videos demoing machine shop techniques

U.S. Suppliers (that ship to Canada & internationally):

  • Little Machine Shop: Parts and supplies for small machine tools
  • Reid Supply: Tooling and automation components
  • Automation Technologies Inc: Automation components
  • Surplus Center: Great source for electric motors and controllers
  • Geckodrive: The best stepper controllers I have ever used
  • eBay: Don't underestimate how useful eBay can be for materials and tooling!
  • SDP/SI: Bearings, Gearing, Belts, Seals, Motors and more.
  • Roton: Great place for buying leadscrews and ballscrews

Canadian suppliers:

  • Applied Industrial Technologies: Bearings, Gearing, Linear motion and more.
  • MotionTek: Servo and Stepper drives, Linear rails, pulleys, vacuum motors and couplers!

Toronto & Ontario suppliers:

  • McKinnon Metals: Carbon and tool steels, aluminum and brass
  • Atlas Machinery: A good selection of tooling, machinery and measurement equipment
  • Machine Tools Warehouse: A great place for machinery and automation controllers
  • Lee Valley: A surprising number of useful precision tools
